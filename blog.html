1. Discribe difference between Local Storage and Session Storage?

Their main difference is that local storage data does not expire, whereas session storage data gets cleared when the user closes its browser window. Page sessions are valid for only one tab at a time. So, if a user has three tabs open, each one will have its own page session. Since local storage is not session-based, all the data must be deleted via javascript or manually.

2. Discribe difference between Global scope and Block scope ?

In a browser environment, the global scope is controlled by the window object while in Node. js, it's controlled by the global object. Block scopes are what you get when you use if statements, for statements, and the like.

4. How can we get undefined in javascript?

A variable that has not been assigned a value is of type undefined . A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned .
